# Smart Search Filter Pattern

Status: **FINAL V1 BASELINE**

Filter harus fleksibel dan searchable.

Flow:

```text
Klik filter
Muncul search
Pilih hasil
Filter aktif menjadi chip
```

Filter penting:
- Produk
- Partner
- Kasir
- Petugas
- Kategori
- Metode Pembayaran
- Jenis Transaksi
- Status
- Nomor Dokumen

Mobile memakai bottom sheet.
Desktop memakai filter bar.
