# Confirmation Modal Pattern

Status: **FINAL V1 BASELINE**

Aksi sensitif wajib modal konfirmasi:

- Batalkan transaksi
- Stock opname posting
- Adjustment
- Reset password
- Ubah permission
- Restore backup

Modal berisi:
- Judul aksi
- Dampak aksi
- Data terdampak
- Alasan jika wajib
- Tombol batal
- Tombol konfirmasi
