# Database Principles

Status: **FINAL V1 BASELINE**

Prinsip:

- Satu database fisik.
- Domain WARUNG / BRILINK / GLOBAL.
- UUID internal.
- Nomor dokumen WRG/BRI.
- Ledger append-only.
- Stock movement untuk stok.
- Snapshot transaksi.
- Soft delete master data.
- Audit Log wajib.
- Atomic transaction.
- Closing snapshot.
- As-Closed dan Adjusted.

Nominal uang disimpan sebagai integer.
