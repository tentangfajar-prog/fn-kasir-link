# PRD V1 — FN Kasir Link

Status: **BASELINE FINAL V1**

## Overview

FN Kasir Link adalah aplikasi internal untuk mengelola Warung dan BRILink dalam satu dashboard Owner dengan pemisahan domain penuh.

## Goals

- POS Warung mobile-first.
- Transaksi BRILink desktop-first.
- Keuangan Warung/BRILink terpisah.
- Closing dua domain.
- Laporan Keuangan untuk matching Owner.
- Audit dan permission kuat.

## Modules

```text
Dashboard
Warung
BRILink
Laporan Keuangan
Absensi
Pengaturan
```

## Non Goals

PPOB, payroll, multi-cabang, native app, integrasi bank, rekonsiliasi otomatis non-tunai.
