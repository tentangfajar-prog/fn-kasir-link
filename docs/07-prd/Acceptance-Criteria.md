# Acceptance Criteria V1

Status: **BASELINE FINAL V1**

Acceptance criteria utama:

- Kasir Warung tidak bisa melihat BRILink.
- Petugas BRILink tidak bisa melihat Warung.
- Owner melihat semua.
- POS tidak bisa dipakai sebelum Kas Laci Awal.
- POS menolak stok kosong/komponen kosong.
- POS tunai menghitung kembalian.
- BRILink menolak Cash/Saldo tidak cukup.
- Fee BRILink otomatis dari master tarif.
- Closing Warung menyimpan selisih dan Setoran Aman.
- Closing BRILink menyimpan selisih Cash/Saldo.
- Laporan Keuangan menampilkan matching Warung/BRILink.
- Chart dashboard tampil harian dan bulanan.
- Audit Log mencatat aksi sensitif.
- Semua transaksi penting atomic.
