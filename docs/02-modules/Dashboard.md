# Dashboard Module

Status: **FINAL V1 BASELINE**

Dashboard memiliki 3 tab:

```text
Ringkasan Utama
Warung
BRILink
```

## Ringkasan Utama

Menampilkan:
- Total Pendapatan Operasional
- Total Profit Operasional
- Omzet Warung
- Pendapatan BRILink
- Total Aset Likuid
- Total Kewajiban
- Alert penting
- Chart harian dan bulanan

## Chart

- Line chart harian bulan ini vs bulan lalu.
- Bar chart bulanan Januari–Desember tahun berjalan.
- Mode gabungan dan pisah Warung/BRILink.
- Chart bisa diklik ke laporan detail.

## Catatan

BRILink tidak dihitung sebagai omzet dari nominal transaksi.
Pendapatan BRILink = Fee Net + Profit Konsinyasi BRILink.
