# Glossary — FN Kasir Link

Status: **FINAL V1 BASELINE**

## Istilah

**Kas Laci**  
Uang tunai fisik Warung yang ada di laci untuk operasional harian.

**Kas Aman**  
Uang Warung yang diamankan oleh Owner. Tetap milik Warung.

**Setoran Aman**  
Perpindahan Kas Laci ke Kas Aman setelah closing. Bukan pengeluaran.

**Cash BRILink**  
Uang fisik BRILink.

**Saldo BRILink**  
Gabungan saldo non-cash/rekening/aplikasi BRILink. Tidak dipisah per rekening di V1.

**Dana di Luar**  
Uang BRILink yang dipinjam keluar sementara dan harus kembali.

**Injeksi**  
Dana tambahan sementara masuk ke BRILink dan harus dikembalikan.

**HPP**  
Harga Pokok, dihitung dengan rata-rata tertimbang untuk stok milik sendiri.

**Konsinyasi**  
Barang titipan. Bukan aset usaha. Kewajiban muncul saat barang laku.

**Closing**  
Laporan penutupan operasional harian setelah cutoff.

**As-Closed**  
Kondisi laporan saat closing disimpan.

**Adjusted**  
Kondisi setelah koreksi/pembatalan setelah closing.

**Reversal**  
Transaksi pembalik untuk membatalkan transaksi tanpa menghapus data asli.
